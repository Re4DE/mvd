# Feature Showcases

The showcases demonstrate the basic usage of the Re4DE Stack without use cases. 

## [Decentralized Catalog](02-features/02-01-catalog/README.md)

With the `Decentralized Catalog`, we demonstrate the general use of the `Connector` to exchange data and restrict access using `Policies`.

## [Integration Permission Administrator](02-features/02-02-pa/README.md)

The `Permission Administrator` in the German energy market is responsible for managing permissions for household meter data from a smart meter gateway.
This showcase demonstrates how a `Permission Administrator` implementation can be integrated into the `Connector`. 

## [Integration Smart-Meter PKI](02-features/02-03-sm-pki/README.md)

The `Smart-Meter PKI` is the trust anchor in the regulated part of the German energy market. 
With this showcase, we demonstrate integration with the `Connector` via the `Data Plane`, enabling a brownfield solution to be integrated into dataspace concepts.
